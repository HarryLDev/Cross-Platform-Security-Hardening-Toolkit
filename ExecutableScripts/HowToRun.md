# How to Run the Scripts

## Windows

### Download WindowsSecure.ps1

### Right-click â†’ Run with PowerShell, or run from terminal:

` powershell -ExecutionPolicy Bypass -File .\WindowsSecure.ps1 `

### A file named windows_audit_YYYY-MM-DD_HH-MM-SS.txt will be created in the same directory.

## macOS

### Download MacSecure.sh

### Open Terminal

### Make script executable:

`chmod +x MacSecure.sh`

### Run:

`./MacSecure.sh` or `sh MacSecure.sh`

### A file named macos_audit_YYYY-MM-DD_HH-MM-SS.txt will appear in the same folder.

## Ubuntu Linux

### Download LinuxSecure.sh

### Open Terminal

### Make script executable:

`chmod +x LinuxSecure.sh`

### Run:

`./LinuxSecure.sh`or `sh LinuxSecure.sh`

### A file named ubuntu_audit_YYYY-MM-DD_HH-MM-SS.txt will be created automatically.
